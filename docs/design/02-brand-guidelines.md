# Roosevelt OPS Brand Guidelines

> AI Strategie & Digital Product Development

Source: `docs/company/profile.md`, `docs/branding-questionnaire.md`

---

## Brand Essence

### Mission

Roosevelt helpt organisaties innoveren met AI strategie en bouwt digitale producten die impact maken.

### Vision

Binnen 5 jaar: De go-to partner voor bedrijven die AI strategie willen vertalen naar concrete digitale producten. Een portfolio van succesvolle eigen producten terwijl we selecte klanten helpen hun digitale ambities te realiseren.

### Values

| Value | Description | How It Shows |
|-------|-------------|-------------|
| **Architectuur Eerst** | Solid foundation voor scalable solutions | We bouwen het fundament voordat we verfraaien |
| **Kwaliteit boven Snelheid** | Done right the first time | Geen shortcuts, geen "dat fixen we later" |
| **Transparante Communicatie** | Open over progress, risks, trade-offs | Eerlijke status updates, geen verrassingen |
| **Direct Access** | Geen layers, direct met expert | Geen account managers, direct met de maker |
| **Strategie + Execution** | Geen handoff tussen denker en maker | Zelfde persoon bedenkt en bouwt |

### Brand Personality

Als Roosevelt een persoon was:

| Trait | Description |
|-------|-------------|
| **Senior architect** | Denkt in systemen, ziet het grote plaatje |
| **No-nonsense** | Komt direct to the point, geen bullshit |
| **Betrouwbaar** | Levert wat beloofd is, op tijd |
| **Technisch diep** | Begrijpt de details, niet alleen het oppervlak |
| **Strategisch** | Combineert visie met hands-on uitvoering |

## Logo

### Concept

Logo is in ontwikkeling. Richtlijnen:

| Attribute | Direction |
|-----------|-----------|
| Style | Clean, geometric, modern |
| Typography | Space Grotesk (bold) |
| Color | Midnight (#0A0F1A) primary, Electric (#3B82F6) accent |
| Symbol | Abstract "R" mark of architectural element |
| Variants needed | Full logo, icon mark, monochrome, favicon |

## Voice & Tone

### Brand Voice

| Attribute | We Are | We Are Not |
|-----------|--------|------------|
| **Professioneel** | Senior, onderbouwd, strategic | Formeel, stijf, bureaucratisch |
| **Direct** | To-the-point, helder, geen omwegen | Kort aangebonden, onbeleefd |
| **Technisch** | Diepgang, accurate terminologie | Jargon-zwaar, ontoegankelijk |
| **Zelfverzekerd** | Weten waar we over praten | Arrogant, neerbuigend |

### Tone per Context

| Context | Tone | Example |
|---------|------|---------|
| Website copy | Professioneel + warm | "We bouwen digitale producten die impact maken." |
| Technical docs | Precies + toegankelijk | "De API accepteert JSON payloads met..." |
| Client comms | Direct + transparant | "Status update: feature X is live, Y heeft 2 dagen vertraging door..." |
| Social media | Kennis + persoonlijk | "AI strategie draait niet om tools. Het draait om..." |

## Photography & Imagery

### Style Direction

| Type | Style | Usage |
|------|-------|-------|
| Abstract | Clean geometric patterns, code abstractions | Hero backgrounds, section dividers |
| Workspace | Minimal, focused, professional | About section, case studies |
| Product | Screenshots, UI mockups | Portfolio, case studies |

### Don'ts

- Geen stockfoto's met mensen die naar schermen wijzen
- Geen overmatig blauwe "tech" beelden
- Geen "handshake" corporate beelden
- Geen AI-gegenereerde gezichten

## What We Do / Don't Do

### We Do

- AI strategie trajecten (opportunity mapping, feasibility, roadmaps)
- Digital product development (design, full-stack, cloud, testing)
- Eigen producten bouwen (VetteShirts, H2WAI, VINO12)

### We Don't Do

- Commodity development (standaard websites/apps)
- Maintenance contracten zonder strategische component
- Projecten zonder architecturele uitdaging
- Outsourcing/staffing werk

# ADR-011: Twenty CRM

**Status:** Accepted
**Date:** 2026-02-26
**Deciders:** Roosevelt Digital Team

## Context

Roosevelt Digital heeft een CRM nodig voor het beheren van client-relaties, deals, en contactpersonen. Een CRM is een commodity tool — we willen geen CRM bouwen. De keuze is welke CRM het beste past bij onze behoeften:

- **Data soevereiniteit:** Client data mag niet bij een derde partij SaaS staan die we niet controleren
- **Open-source:** Aanpassingen en integraties mogelijk zonder API-limieten
- **MCP-integratie:** LLM agents moeten CRM data kunnen lezen en schrijven
- **Zelfhost:** Draait op onze eigen Hetzner infrastructuur
- **Moderne UI/UX:** Het team moet er dagelijks mee willen werken
- **Prijs:** Geen per-seat licentiekosten voor klein team

## Decision

We kiezen voor **Twenty CRM** (self-hosted via Docker Compose op Hetzner).

Twenty is een open-source alternatief voor Salesforce en HubSpot, gebouwd op PostgreSQL met een moderne React-based UI. De GraphQL API biedt volledige programmatische toegang.

**Technische stack:**
- Twenty self-hosted via Docker Compose op Hetzner VPS
- PostgreSQL backend (zelfde als Supabase postgres)
- GraphQL API voor integraties
- MCP-server voor LLM agent toegang tot CRM data
- Webhooks voor real-time synchronisatie met Plane en onze integratie layer

**Kerngebruik:**
- Client en contactpersoon management
- Deal pipeline (proposal → contract → won/lost)
- `deal.won` event → Inngest pipeline (zie ADR-007)

**Rationale:**
- Geen per-seat kosten (alleen hosting: ~€20/maand op Hetzner)
- Volledige data controle (GDPR compliance op eigen infra)
- Actief onderhouden open-source project met grote community
- GraphQL API maakt diepe integraties mogelijk
- Modern UI dat het team wil gebruiken

## Consequences

### Positief
- Nul licentiekosten, alleen hosting
- Volledige controle over data en customisatie
- Diepe integratie mogelijk via GraphQL API en webhooks
- GDPR-vriendelijk (data op onze eigen servers)

### Negatief / Trade-offs
- Zelfhostbeheer: updates, backups, monitoring zijn onze verantwoordelijkheid
- Minder volwassen dan Salesforce/HubSpot (features missen soms)
- Community support (geen enterprise SLA)
- Initiële setup en configuratie tijd

## Alternatieven overwogen

- **HubSpot:** Rijke feature set maar per-seat kosten lopen snel op. Data bij HubSpot (vendor lock-in). Geen open API voor diepgaande integraties zonder upgrade. Afgewezen vanwege kosten en data soevereiniteit.
- **Salesforce:** Enterprise-grade maar absoluut overpriced en overkill voor een klein team. Afgewezen.
- **Pipedrive:** Goed voor sales teams maar beperkte customisatie en API-mogelijkheden. Geen self-host optie. Afgewezen vanwege vendor lock-in.
- **Custom CRM:** Volledig bouwen kost 2–4 weken en brengt ons geen concurrentievoordeel. Afgewezen (zie ADR-008, Build vs Buy).

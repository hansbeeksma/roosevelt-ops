# Typography Resources

Gecureerde verzameling gratis fonts en tools voor Roosevelt projecten en concepten.
Alle fonts zijn vrij voor commercieel gebruik (OFL 1.1 / Apache 2.0).

---

## Aanbevolen Font Combinaties

| Gebruik | Font | Stijl | Bron |
|---------|------|-------|------|
| **UI / Interface** | Inter | Geometric sans | [Google Fonts](https://fonts.google.com/specimen/Inter) |
| **Headlines** | Space Grotesk | Grotesque sans | [Google Fonts](https://fonts.google.com/specimen/Space+Grotesk) |
| **Body text** | Source Serif 4 | Editorial serif | [GitHub](https://github.com/adobe-fonts/source-serif) |
| **Code / Mono** | Source Code Pro | Monospace | [Google Fonts](https://fonts.google.com/specimen/Source+Code+Pro) |
| **Alternatief UI** | Manrope | Geometric sans | [Google Fonts](https://fonts.google.com/specimen/Manrope) |

## Gratis Alternatieven voor Premium Fonts

Referentie: [Klim Type Foundry](https://klim.co.nz) premium fonts en hun open-source equivalenten.

| Premium Font | Stijl | Gratis Alternatief | Licentie |
|-------------|-------|-------------------|----------|
| Söhne | Geometric sans-serif | **Inter**, **Manrope** | OFL 1.1 |
| Founders Grotesk | Grotesque sans-serif | **Space Grotesk**, **Overused Grotesk** | OFL 1.1 |
| National 2 | Humanist sans-serif | **Open Sans**, **Alegreya Sans** | OFL 1.1 / Apache 2.0 |
| Tiempos | Editorial serif | **Source Serif 4 Display** | OFL 1.1 |
| Domaine | Display serif | **EB Garamond**, **Lusitana** | OFL 1.1 |

> **Tip:** Source Serif 4 via GitHub downloaden voor Display/Headline weights - Google Fonts heeft alleen de Text variant.
